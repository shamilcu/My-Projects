# Noire Luxury - Running the Project

## Quick Start

This project is a static HTML/CSS/JS site. To view it properly, especially with external assets and modules, you should run it using a local server.

### Option 1: Using Python (Recommended)

1. Open a terminal in the project folder.
2. Run:
   ```bash
   python -m http.server 8080
   ```
3. Open your browser to: [http://localhost:8080](http://localhost:8080)

### Option 2: Using VS Code Live Server

1. Install the "Live Server" extension in VS Code.
2. Right-click `index.html` and select "Open with Live Server".

## Project Structure

- `index.html`: Main landing page
- `style.css`: Stylesheets
- `script.js`: Core logic and animations
- `threeScene.js`: 3D scene logic (if used)
